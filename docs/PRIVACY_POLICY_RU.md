# Политика конфиденциальности

**Последнее обновление: март 2026**

---

## 1. Введение

Настоящая Политика конфиденциальности описывает, как приложение **«План»** (далее — «Приложение», «мы») обрабатывает информацию пользователей. Приложение разработано для личного некоммерческого использования.

**Важно:** Мы не собираем, не храним и не передаём никакие персональные данные пользователей.

---

## 2. Информация, которую мы НЕ собираем

Приложение **НЕ** собирает, не хранит и не передаёт:

- ✅ Персональные данные пользователя
- ✅ Имя, адрес электронной почты, номер телефона
- ✅ Геолокацию пользователя
- ✅ Данные о поведении пользователя
- ✅ Информацию об устройстве
- ✅ Статистику использования приложения
- ✅ Фотографии и видео пользователей (они хранятся только на устройстве пользователя)

---

## 3. Где хранятся данные

Все данные, создаваемые в Приложении, хранятся **исключительно на устройстве пользователя**:

| Тип данных | Место хранения |
|------------|----------------|
| Проекты | Локальная база данных SQLite (Room) |
| Области (регионы) | Локальная база данных SQLite (Room) |
| Фотографии | Внутренняя память приложения (`/data/data/com.plan.app/files/media/`) |
| Видео | Внутренняя память приложения (`/data/data/com.plan.app/files/media/`) |
| Настройки | DataStore (локальное хранилище) |

**Данные НЕ передаются** на внешние серверы, в облако или третьим лицам.

---

## 4. Удаление данных

При удалении Приложения все данные **автоматически удаляются** с устройства. Никакие данные не остаются на серверах, так как мы не используем серверы.

---

## 5. Разрешения приложения

Приложение запрашивает следующие разрешения:

| Разрешение | Цель | Обязательно |
|------------|------|-------------|
| Камера | Съёмка фото и видео для проектов | Да |
| Доступ к медиафайлам | Импорт фото и видео из галереи | Да |

**Разрешения, запрошенные для будущих функций (временно):**

| Разрешение | Цель | Статус |
|------------|------|--------|
| Интернет | Будущие функции синхронизации | Не используется |
| Wi-Fi Direct | Передача данных между устройствами | Не используется |
| Bluetooth | Передача данных между устройствами | Не используется |

---

## 6. Реклама

Приложение **НЕ содержит рекламы**:
- ✅ Нет рекламных баннеров
- ✅ Нет видео-рекламы
- ✅ Нет всплывающей рекламы
- ✅ Нет рекламных SDK (Google AdMob, Facebook Ads и др.)
- ✅ Нет трекеров аналитики

---

## 7. Сторонние библиотеки

Приложение использует следующие библиотеки с открытым исходным кодом:

| Библиотека | Назначение |
|------------|------------|
| Android Jetpack | UI компоненты |
| Room | Локальная база данных |
| Hilt | Внедрение зависимостей |
| Coil | Загрузка изображений |
| ExoPlayer | Воспроизведение видео |
| Gson | Сериализация данных |

Ни одна из этих библиотек **не передаёт данные** третьим лицам.

---

## 8. Безопасность

Приложение реализует следующие меры безопасности:

- ✅ Локальное хранение данных (Scoped Storage)
- ✅ Отключено резервное копирование данных
- ✅ Обфускация кода в release-сборке
- ✅ Автоматическая очистка временных файлов

---

## 9. Возрастные ограничения

Приложение подходит для пользователей любого возраста. Мы не собираем данные, в том числе от несовершеннолетних пользователей.

---

## 10. Изменения политики

Мы можем обновлять настоящую Политику конфиденциальности. О существенных изменениях будет сообщено через обновление приложения.
