## Разрешения библиотек

| Библиотека | Требуемые разрешения |
|------------|---------------------|
| CameraX | android.permission.CAMERA |
| Coil | android.permission.INTERNET (для загрузки из сети — не используется) |
| Room | Нет |
| Hilt | Нет |
| DataStore | Нет |

**Примечание:** Разрешение INTERNET объявлено в манифесте для будущих функций, но в текущей версии **не используется ни одной библиотекой**.

Приложение проверено на наличие трекеров с помощью инструментов анализа APK:

- **App Manager:** Трекеры не обнаружены
- **ClassyShark:** Трекеры не обнаружены

## Заключение

Приложение «План» является полностью локальным приложением без сетевой активности. Все данные хранятся и обрабатываются исключительно на устройстве пользователя.
