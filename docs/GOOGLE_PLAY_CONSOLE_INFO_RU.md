## Разрешения и их обоснование

| Разрешение | Обоснование для Google Play |
|------------|----------------------------|
| `android.permission.CAMERA` | Необходимо для съёмки фото и видео, которые пользователь хочет аннотировать |
| `android.permission.READ_MEDIA_IMAGES` | Для импорта изображений из галереи устройства (Android 13+) |
| `android.permission.READ_MEDIA_VIDEO` | Для импорта видео из галереи устройства (Android 13+) |
| `android.permission.READ_EXTERNAL_STORAGE` (maxSdkVersion=32) | Для поддержки устройств Android 12 и ниже |
| `android.permission.WRITE_EXTERNAL_STORAGE` (maxSdkVersion=28) | Для поддержки устройств Android 9 и ниже |

```
Это приложение является некоммерческим проектом с открытым исходным кодом.

Приложение НЕ содержит:
- Рекламы (баннеров, видео, всплывающих окон)
- Покупок внутри приложения
- Сбор персональных данных
- Аналитики и трекеров
- Связи с внешними серверами
- Интернет-соединения

Все данные хранятся локально на устройстве пользователя.

Исходный код доступен на GitHub:
https://github.com/dvol123/plan_v1
```

## ✅ Минимальный набор разрешений

Приложение использует только необходимые для работы разрешения:

1. **CAMERA** — для съёмки фото и видео
2. **READ_MEDIA_IMAGES** — для импорта изображений
3. **READ_MEDIA_VIDEO** — для импорта видео

Никаких дополнительных разрешений не требуется.
