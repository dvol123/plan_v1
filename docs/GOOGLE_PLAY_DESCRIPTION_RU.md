# Описание приложения для Google Play Store

---

## Название приложения
План

---

## Краткое описание (80 символов)
Аннотируйте фото областями. Добавляйте фото, видео и описания.

---

## Полное описание

**План** — приложение для создания интерактивных аннотаций на фотографиях.

Идеально подходит для:
• Планирования помещений и территорий
• Документации дефектов и проверок
• Маркировки областей на картах и схемах
• Организации фотоотчётов

### Основные возможности:

📷 **Создание проектов**
— Добавляйте фото из галереи или делайте снимки прямо в приложении
— Присваивайте проектам понятные названия

🎯 **Выделение областей**
— Рисуйте области на фото с помощью сетки
— Настраивайте размер сетки для точности
— Выделяйте несколько областей на одном фото

🏷️ **Аннотации и состояния**
— Добавляйте названия и описания к каждой области
— Присваивайте цветовые состояния (выполнено, в процессе, ожидание)
— Создавайте собственные категории

📸 **Медиа-контент**
— Прикрепляйте фото к каждой области
— Добавляйте видео с комментариями
— Просматривайте контент в полноэкранном режиме

📤 **Экспорт данных**
— Экспортируйте проекты в HTML для просмотра на ПК
— Создавайте ZIP-архивы для резервного копирования
— Передавайте проекты между устройствами

💾 **Локальное хранение**
— Все данные хранятся только на вашем устройстве
— Никаких облачных сервисов и подписок
— Полный контроль над вашими данными

### Особенности:
✓ Без рекламы
✓ Без подписок
✓ Без сбора данных
✓ Работает офлайн
✓ Современный Material Design 3
✓ Поддержка Android 14+
